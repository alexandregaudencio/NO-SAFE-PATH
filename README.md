# NO-SAFE-PATH - Chaotic Escape Multiplayer

Chaotic multiplayer escape game where players must survive constant pursuit by enemies, using the environment strategically to evade, delay, and outmaneuver threats.

Combat is secondary; positioning, timing, and spatial awareness are the core mechanics.

Built with Unity and Photon Fusion, targeting mobile platforms, with a strong emphasis on modular architecture, clean code, and scalability.
